CSS

the design,layout and variations in display for different devices and screen sizes. Stands for "Cascading Style Sheet." Cascading style sheets are used to format the layout of Web pages.

They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML.
